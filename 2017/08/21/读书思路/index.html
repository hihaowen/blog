读书思路

Posted on 2017-08-21

探索(整个时间的 5% ~ 10%)

1. 观点是什么？
2. 方法是什么？
3. 结论是什么？
4. 标记重点

理解(整个时间的70%~80%)

1. 什么论据来支撑观点、方法、结论？
2. 这些观点、论证和结论的缺陷是什么？
3. 你怎么认为这些观点、证据和结论？
4. 作者怎样（或者怎么能）回应这些缺陷，和你的批评？

回忆和记笔记

1. 想象你正在和作者进行一个对话，或者是一个普通的辩论
2. 想象给书一个考试。考试的题目是什么，你是怎样回答的？
    → 记录重要元素，更新到自己理解框架
    → 平均每 100 页输出 1 - 3 页笔记
    → 笔记可以快速的定位到书中内容
